## 📊 优先级总览

| 优先级 | 数量 | 说明 |
|--------|------|------|
| P0 紧急重要 | 2 | 用户刚需，强烈推荐实现 |

## 🚨 P0 - 紧急重要（立即实现）

### 1. 深色模式

**功能描述**：支持系统深色模式跟随，手动切换深浅主题

**用户价值**：
- 夜间使用护眼，减少屏幕刺眼感
- 符合用户使用习惯（大量用户在晚间使用）

**实现思路**：
- 使用 TailwindCSS 的 dark 模式
- 检测系统 prefers-color-scheme
- 提供手动切换按钮

**验收标准**：
- [x] 跟随系统深色/浅色模式切换
- [x] 手动切换后记忆用户选择
- [ ] 所有页面颜色正常显示

---

### 2. 快捷场景筛选

**功能描述**：首页增加快捷场景筛选项：快手菜、下饭菜、荤菜、素菜、汤类、早餐等

**用户价值**：
- 快速定位目标菜品
- 解决"不知道吃什么"的困扰

**实现思路**：
- 首页分类tab改为横向滚动筛选
- 预设常见场景标签
- 支持自定义筛选组合

**验收标准**：
- [ ] 首页展示快捷场景入口
- [ ] 点击快速筛选对应菜品
- [ ] 支持多条件组合筛选
- [ ] 筛选后显示匹配数量

## ⭐ P1 - 重要不紧急（计划实现）

### 1. 今日吃什么骰子

**功能描述**：家庭成员各自选3道想吃的菜，点击"掷骰子"随机选一道

**用户价值**：
- 解决选择困难户的痛点
- 增加家庭互动仪式感

**实现思路**：
- 创建"骰子模式"页面
- 家庭成员各自添加候选菜品
- 随机抽取，生成结果

**验收标准**：
- [ ] 支持家庭成员各自添加候选
- [ ] 掷骰子动画效果
- [ ] 显示中选菜品详情
- [ ] 可直接加入购物车

---

### 2. 家庭成员切换

**功能描述**：支持多家庭成员独立口味档案，一键切换视图

**用户价值**：
- 多成员家庭必备功能
- 各成员有独立推荐和收藏

**实现思路**：
- 用户系统增加家庭成员管理
- 每个成员独立口味偏好
- 一键切换当前使用者

**验收标准**：
- [ ] 添加/删除家庭成员
- [ ] 每个成员独立口味设置
- [ ] 切换后推荐相应菜品
- [ ] 收藏夹独立存储

---

### 3. 厨艺勋章系统

**功能描述**：完成菜品解锁成就勋章（新手上路、连煮7天、甜品达人等）

**用户价值**：
- 游戏化激励，增加做菜动力
- 成就感驱动，形成习惯

**实现思路**：
- 设计勋章体系（青铜、白银、黄金）
- 定义成就触发条件
- 勋章展示页面

**验收标准**：
- [ ] 显示已获得勋章
- [ ] 进度条显示解锁进度
- [ ] 获得勋章动画效果
- [ ] 分享勋章到社交平台

---

### 4. 图片懒加载 + 骨架屏

**功能描述**：图片懒加载，首屏加载显示骨架屏

**用户价值**：
- 提升首屏加载速度
- 减少等待焦虑

**实现思路**：
- Vue Lazy Load 组件
- 首屏关键资源预加载
- 骨架屏组件设计

**验收标准**：
- [ ] 图片进入可视区域才加载
- [ ] 首屏显示骨架屏
- [ ] 图片渐进式加载动画
- [ ] 离线缓存图片

---

### 5. 新手引导

**功能描述**：首次使用引导设置口味偏好、家庭成员、烹饪水平

**用户价值**：
- 帮助快速上手
- 个性化推荐基础

**实现思路**：
- 设计引导流程（ onboarding）
- 分步收集用户信息
- 跳过/下一步交互

**验收标准**：
- [ ] 首次打开进入引导流程
- [ ] 设置口味偏好
- [ ] 添加家庭成员
- [ ] 完成后展示个性化推荐

---

### 6. 步骤进度条

**功能描述**：食谱步骤实时显示当前进度和完成百分比

**用户价值**：
- 降低烹饪焦虑
- 清晰了解整体流程

**实现思路**：
- 步骤组件增加进度条
- 当前步骤高亮标识
- 完成百分比计算

**验收标准**：
- [ ] 进度条实时更新
- [ ] 当前步骤高亮
- [ ] 步骤可点击跳转
- [ ] 完成后显示庆祝动画

---

### 7. 收藏夹分类管理

**功能描述**：收藏夹支持自定义分类（周末做、请客菜、宝宝爱吃）

**用户价值**：
- 整理自己的菜单库
- 便于快速找到目标菜品

**实现思路**：
- 收藏页面增加分类tab
- 支持创建自定义分类
- 收藏时选择分类

**验收标准**：
- [ ] 默认分类 + 自定义分类
- [ ] 收藏时选择分类
- [ ] 分类内支持排序
- [ ] 一键移动分类

---

### 8. 营养成分展示

**功能描述**：展示菜品热量、蛋白质、维生素等营养成分

**用户价值**：
- 健康饮食参考
- 健身/减肥人群刚需

**实现思路**：
- 菜品数据增加营养字段
- 食谱详情页展示营养标签
- 筛选功能增加营养维度

**验收标准**：
- [ ] 菜品详情显示营养成分
- [ ] 标签展示（低脂、高蛋白等）
- [ ] 筛选支持营养维度
- [ ] 日/周营养摄入统计

### 7. 忌口/过敏设置

**功能描述**：设置过敏原（花生、海鲜等）、不吃香菜/葱姜蒜等，系统自动过滤并提醒

**用户价值**：

### 8. 语音播报步骤

**功能描述**：语音播报当前步骤，支持"下一步"、"再读一遍"、"计时开始"等语音指令

**用户价值**：
- 解放双手，炒菜时无需触碰手机
- 高频使用场景，实用性强

---

### 9. 购物清单导出PDF

**功能描述**：生成PDF格式购物清单，支持打印

**用户价值**：
- 超市采购实用工具
- 纸质清单便于勾选核对

## 📝 交互设计创意讨论（进行中）

> 由 UX 创意团队讨论生成

### 1. 首页 - 像刷小红书一样刷菜谱

**方案**：瀑布流卡片 + 手势交互
- 卡片右滑=收藏，左滑=跳过（像Tinder）
- 顶部"今天吃啥"按钮直接进入骰子模式
- 横向滑动标签（快手菜、下饭菜等）像抖音话题

### 2. 菜品卡片 - 像发朋友圈一样分享

**方案**：小红书风格卡片设计
- 大图预览，点击放大
- 操作按钮动效：点击收藏❤️弹出消失、点击加购🛒弹跳
- 长按弹出快捷菜单

### 3. 购物车 - 超市采购模式

**方案**：交互式采购清单
- 左滑删除，右滑标记已购
- 语音播报："五花肉500g"
- 预算进度条实时显示

### 4. 食谱页面 - 像玩游戏一样做饭

**方案**：游戏化步骤设计
- 大图当前步骤 + 进度条
- 完成步骤✅勾选动画+轻微震动
- 计时器番茄钟样式
- 语音模式波形动画

### 5. 趣味骰子模式

**方案**：物理抛掷动画
- 骰子有抛掷动画
- 结果弹出彩带/星星动画
- "撒娇求饶"任务抵消

### 6. 手势操作汇总

| 手势 | 位置 | 动作 |
|------|------|------|
| 上滑 | 菜品卡片 | 查看详情 |
| 左滑 | 菜品卡片 | 加入购物车 |
| 右滑 | 菜品卡片 | 收藏 |
| 长按 | 任意 | 弹出快捷菜单 |
| 摇一摇 | 首页 | 随机选菜 |
| 双指缩放 | 图片 | 放大缩小 |
